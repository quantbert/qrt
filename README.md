# Quant Research Tools (qrt)

QRT is an umbrella library of quantitative research tools: one consistent API over the fragmented quant Python ecosystem (data access, feature engineering, model utilities, backtesting, portfolio analysis, return statistics, and plotting).

## Warning ⚠️

Library is still under early development. Come back later.

## Install

```bash
uv add pyqrt
```

## Docs

Full documentation, API reference, and roadmap: https://quantbert.github.io/qrt/
