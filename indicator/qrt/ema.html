Exponential moving average (EMA)

q.indicator.ema calculates an exponentially weighted moving average with adjust=False, so recent observations receive more weight than older observations. It accepts one ordered Series and preserves its index.

This example compares 50-session EMAs for AAPL and the SPY benchmark over the latest five shared years in qrt’s bundled data.

import pandas as pd

import qrt as q

prices = pd.concat(
    {
        "AAPL": q.data.datasets.load("aapl")["close"],
        "SPY": q.data.datasets.load("spy")["close"],
    },
    axis=1,
    join="inner",
).dropna()
end = prices.index.max()
prices = prices.loc[end - pd.DateOffset(years=5) :]
prices.tail()

Calculate the indicator

Call ema separately for each instrument. A positive integer window is interpreted as the exponential span.

window = 50
averages = prices.apply(lambda series: q.indicator.ema(series, window))
averages.tail()

Compare with SPY

Each asset is indexed to 100 at the beginning of the window. The EMA uses the same denominator as its underlying close, preserving the distance between price and average.

base = prices.iloc[0]
comparison = pd.concat(
    [
        prices.div(base).mul(100).add_suffix(" close"),
        averages.div(base).mul(100).add_suffix(f" EMA {window}"),
    ],
    axis=1,
)
fig = q.plot.col(
    comparison,
    title=f"AAPL and SPY with {window}-session exponential moving averages",
    ylabel="Indexed value (start = 100)",
)
fig.show(renderer="notebook_connected")
