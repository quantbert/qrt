Mean absolute deviation from SMA (MADEV)

q.indicator.madev measures the trailing mean absolute distance between one instrument’s close and its simple moving average. The reviewed formula uses the same window twice, so the first defined value requires \(2n - 1\) observations.

The function accepts one ordered Series, preserves its index, and does not mutate the input.

import pandas as pd

import qrt as q

data = q.data.datasets.load("aapl")
end = data.index.max()
data = data.loc[end - pd.DateOffset(years=5) :].copy()
window = 20
data["madev"] = q.indicator.madev(data["close"], window=window)
data[["close", "madev"]].tail()
