Relative-strength phase

q.indicator.rs_phase flags dates when relative strength is above a supplied moving average and counts consecutive dates in that state. It returns both rs_phase and rs_phase_days.

AAPL relative strength is measured against SPY, then compared with its 21-session RSMA over the latest five shared years.

import pandas as pd

import qrt as q

prices = pd.concat(
    {
        "AAPL": q.data.datasets.load("aapl")["close"],
        "SPY": q.data.datasets.load("spy")["close"],
    },
    axis=1,
    join="inner",
).dropna()
end = prices.index.max()
prices = prices.loc[end - pd.DateOffset(years=5) :]
strength = q.indicator.relative_strength(prices["AAPL"], prices["SPY"], lookback=63)
average = q.indicator.rsma(strength, window=21)
average.tail()

Calculate the indicator

The moving average is supplied explicitly, so callers can choose its method and window before phase classification.

phase = q.indicator.rs_phase(strength, average)
phase.tail()

Compare with SPY

Relative strength and RSMA are measured in percentage points against SPY. The phase-length line resets to zero whenever relative strength is not above its average.

comparison = pd.DataFrame(
    {
        "AAPL relative strength vs SPY (%)": strength.mul(100),
        "AAPL RSMA (%)": average.mul(100),
        "Phase length (sessions)": phase["rs_phase_days"],
    }
)
fig = q.plot.col(
    comparison,
    title="AAPL relative-strength phase versus SPY",
    ylabel="Percentage points / sessions",
)
fig.show(renderer="notebook_connected")
