Roadmap

Ideas and planned work per submodule. Checked items are done, unchecked are planned.

General

q.stats — return-stream statistics

Stateless analytics for periodic return streams (no plotting dependency), plus an optional bound q.stats.returns(...) object for chaining stats and plots. Consumed by q.plot for rendering.

q.plot — plotting with premade quant-finance configs

Opinionated plotting helpers with sensible defaults for quant research.

Inspiration

Libraries we take inspiration from (and in some cases wrap):

Library What we borrow
quantstats tearsheets, return-stream metrics
pyfolio-reloaded portfolio/performance analysis
alphalens-reloaded alpha-factor evaluation
empyrical-reloaded risk/performance statistics
tsfresh automated time-series feature extraction
skfolio sklearn-style portfolio optimization
mlfinlab purged CV, embargo, financial ML (López de Prado)
pandas-ta technical indicators
pandas-ta-classic maintained fork of pandas-ta indicators
Riskfolio-Lib portfolio optimization & risk measures
qlib end-to-end quant ML platform design
pytorch-forecasting PyTorch time-series model wrappers
sktime unified time-series API design
alphatools alpha research & factor tooling on a securities master
PyStats statistical distribution functions (pdf/cdf/quantile/sampling)
